# 命令行

## new

`new`命令用来创建一个新的解决方案，与使用`Studio`图形化界面创建是相同的效果。

```pwsh
ater new <name>
```

创建解决方案的流程：

1. 选择项目类型: 标准或轻量
2. 选择数据库类型：Sqlserver、PostgreSql
3. 输入连接字符串
4. 选择缓存类型
   1. 不是内存时，输入连接字符串
5. 选择队列类型(暂不支持)
6. 选择授权类型(暂不支持)
7. 其他配置(同时选中多个，暂不支持)
8. 模块选择(多选)
9. 输入目录，默认为当前
10. 确认并运行创建

## studio

`studio`命令用来启动图形化界面

```pwsh
ater studio
```
